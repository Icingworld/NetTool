## 以太网切换工具

### 功能

+ 修改网络配置信息
+ 检测当前网络配置
+ ping检测以太网设备连接
+ 历史记录

## 注意事项

+ `set.txt`中为以太网适配器名字，默认为“以太网 3”

+ 如需要改成自己的，`win+r`，输入`cmd`，运行`ipconfig`查看适配器名字，点击`检测`查看信息

+ 需要以管理员身份运行
